CloudWatch APM Documentation (Static Site)

This repository contains a static documentation site scaffold for CloudWatch Application Performance Monitoring (APM).

Quick start (serve locally):

```bash
cd /Users/vijaykbv/Documents/Projects/CW-APM-Docs/docs
python3 -m http.server 8000
# then open http://localhost:8000
```

The site content is under `docs/` and is generated as plain HTML/CSS/JS for easy hosting.
