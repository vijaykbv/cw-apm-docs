CloudWatch APM Documentation

Integrations

AWS Service Integrations

CloudWatch Application Signals integrates seamlessly with AWS services to provide comprehensive observability across your entire cloud infrastructure.

API Gateway

Monitor API performance and track requests from API Gateway to your backend services.

Integration Features:

  • Automatic tracing of API calls
  • Latency monitoring for API endpoints
  • Error rate tracking
  • Request/response correlation

Setup:

Enable X-Ray tracing in your API Gateway stage configuration.

Application Load Balancer (ALB)

Track traffic flow through load balancers and correlate with application performance.

Integration Benefits:

  • Request routing visibility
  • Target group performance metrics
  • Health check monitoring
  • Load distribution analysis

DynamoDB

Monitor database operations and identify performance bottlenecks in data access patterns.

Tracing Capabilities:

  • Query operation timing
  • Throttle and error tracking
  • Table and index performance
  • Cross-region replication monitoring

Amazon RDS

Get insights into database performance and connection pooling.

Database Monitoring:

  • Query execution times
  • Connection pool utilization
  • Database instance metrics
  • Slow query identification

Amazon S3

Track object storage operations and data transfer performance.

Storage Operations:

  • Put/Get object tracing
  • Transfer latency monitoring
  • Error rate analysis
  • Cross-region transfer tracking

SNS/SQS

Monitor message processing and queue performance in event-driven architectures.

Messaging Insights:

  • Message publish/subscribe tracing
  • Queue depth monitoring
  • Message processing latency
  • Dead letter queue analysis

AWS Step Functions

Visualize workflow execution and identify bottlenecks in state machine processing.

Workflow Monitoring:

  • State transition tracing
  • Execution time analysis
  • Error state identification
  • Parallel execution monitoring

Amazon EventBridge

Track event routing and processing through event-driven systems.

Event Processing:

  • Event routing visibility
  • Rule evaluation timing
  • Target invocation tracing
  • Event transformation monitoring

AWS Lambda

Monitor serverless function performance and cold start impacts.

Serverless Insights:

  • Function invocation tracing
  • Cold start duration
  • Memory and CPU utilization
  • Error and timeout tracking

Amazon ECS/EKS

Container orchestration platform integration for microservices monitoring.

Container Monitoring:

  • Service mesh tracing
  • Pod-to-pod communication
  • Resource utilization
  • Deployment tracking

Custom Integrations

Build your own integrations using APIs and SDKs.

CloudWatch APIs

Use CloudWatch APIs to build custom integrations and dashboards.

Available APIs:

  • PutMetricData - Send custom metrics
  • GetMetricData - Retrieve metric data
  • PutTraceSegments - Send trace data
  • GetTraceSummaries - Query trace data

Webhooks and Callbacks

Receive real-time notifications and integrate with custom workflows.

Integration Patterns:

  • Alert webhooks for custom notification systems
  • Event-driven processing with Lambda
  • Custom dashboard updates
  • Automated remediation workflows
